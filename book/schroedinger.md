# Persamaan Schrödinger

```{prf:definition} Persamaan Schrödinger
:label: schroedinger

:::::{tab-set}
::::{tab-item} 1 Dimensi
Pertimbangkan suatu zarah berfungsi gelombang $\Psi$ yang merambat dalam medan keupayaan $V(x)$, maka sifat zarah tersebut tertakluk pada persamaan berikut:

$$
    \frac{-\hbar^2}{2m}\frac{\partial^2}{\partial^2 x}\Psi(x) + V(x)\Psi(x) = i\hbar\frac{\partial}{\partial t}\Psi(x)
$$

% senarai pemalar
:::{dropdown} dengan maksud bahawa...

| simbol | maksud | nilai |
| :---: | :--- | :--- |
| $\hbar$ | pemalar Planck palang | $1.054571817...\times 10^{−34}$ J$\cdot$s |
| $m$ | jisim zarah | pembolehubah tidak bersandar |
| $x$ | kedudukan | pembolehubah tidak bersandar |
| $V(x)$ | keupayaan | fungsi bersandar $x$ |
| $\Psi(x)$ | fungsi gelombang | fungsi bersandar $x$ |

:::
::::
::::{tab-item} 3 Dimensi
Pertimbangkan suatu zarah berfungsi gelombang $\Psi$ yang merambat dalam medan keupayaan $V(r)$, maka sifat zarah tersebut tertakluk pada persamaan berikut:

$$
    \frac{-\hbar^2}{2m}\nabla^2\Psi(r) + V(r)\Psi(r) = i\hbar\frac{\partial}{\partial t}\Psi(r)
$$

% senarai pemalar
:::{dropdown} dengan maksud bahawa...

| simbol | maksud | nilai |
| :---: | :--- | :--- |
| $\hbar$ | pemalar Planck palang | $1.054571817...\times 10^{−34}$ J$\cdot$s |
| $m$ | jisim zarah | pembolehubah tidak bersandar |
| $r$ | kedudukan dalam 3 dimensi | pembolehubah tidak bersandar |
| $\nabla^2$ | operator Laplace | $\frac{\partial^2}{\partial^2 x} + \frac{\partial^2}{\partial^2 y} + \frac{\partial^2}{\partial^2 z}$ |
| $V(r)$ | keupayaan | fungsi bersandar $r$ |
| $\Psi(r)$ | fungsi gelombang | fungsi bersandar $r$ |

:::
::::
```
